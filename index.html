What is Computer Memory?

Computer Memory is a physical device located inside the computer, that is capable of storing information temporarily (RAM) (random access memory) or permanently (ROM) (read-only memory). Both of these devices utilize integrated circuits that are used by the operating systems, the hardware and the software of a computer.

Volatile vs. Non-volatile Computer Memory

Computer Memory can be either volatile or non-volatile. Volatile memory loses its contents when the computer or hardware is powered off. Computer (RAM) is an example of volatile memory, if your computer turns off while you are working on a document saved in the (RAM), you will lose anything that hasn't been saved on that document. Non-volatile memory is the opposite, it does not lose its contents, even when abruptly powered off. (ROM) is an example of non-volatile memory. This video explains more about volatile and non-volatile memory.

Computer Memory is not disk storage

New computer users might be confused by what parts of the computer are memory. Although both the hard drive and (RAM) are forms of memory, it is more fitting to refer to (RAM) as "memory" or "primary memory" and the hard drive as "storage" or "secondary memory".
