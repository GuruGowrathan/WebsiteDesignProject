RAM (random-access memory)

Types of RAM

Over the years and the evolution of the computer, there have been different variations of RAM. Some of the more common examples include DIMM, RIMM, SIMM, SO-DIMM, and SOO-RIMM. Additional RAM information

As the computer starts up, parts of the operating system and drivers are loaded into memory, which allows the CPU (central processing unit) to process the instructions faster and speeds up the boot process. After the operating system has started up, each program you open, such as the browser you're using to view this page, is loaded into memory while it is running. If too many programs are open, the computer will swap the data in the memory between the RAM and the hard disk drive.

The performance and speed of a computer is largely attributed to the quantity of memory that can be stored within it. If a computer does not have enough memory to run the operating system and the software programs, it results in slower performance. In short, the more memory a computer has, the more information and software it can load and process quicker.
